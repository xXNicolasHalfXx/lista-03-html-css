# 🌐 Layout Web Responsivo

## 📌 Objetivo

Desenvolver uma página web responsiva utilizando HTML e CSS, aplicando conceitos fundamentais como viewport, media queries, breakpoints, Flexbox e CSS Grid.

---

## 💡 Descrição do Projeto

O projeto consiste em uma página institucional temática inspirada no personagem Grogu, contendo:

* Header com logo e menu de navegação
* Seção principal com cards informativos
* Footer simples

O layout foi desenvolvido para se adaptar a diferentes tamanhos de tela, garantindo boa usabilidade em dispositivos desktop, tablet e mobile.

---

## 🛠️ Técnicas Utilizadas

### 🔹 CSS Grid

Utilizado para estruturar o layout principal da página e organizar os cards em colunas no desktop.

### 🔹 Flexbox

Aplicado no menu de navegação e no alinhamento interno dos cards.

---

## 📱 Responsividade

A responsividade foi implementada utilizando media queries com os seguintes breakpoints:

* **Desktop:** acima de 900px (layout com múltiplas colunas)
* **Tablet:** até 900px (redução para 2 colunas)
* **Mobile:** até 600px (layout em coluna única e menu vertical)

---

## 🎨 Estilização

* Uso de CSS externo
* Aplicação de reset de CSS
* Uso de `box-sizing: border-box`
* Paleta de cores em tons de verde suave
* Utilização de imagens para enriquecer os cards
